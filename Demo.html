Visualizing the Distributions

We can generate various plots like histogram and box plot to visualize the distributions of variables in a data set

Histogram

hist(mtcars$mpg,
     main = "Histogram of MPG",
     xlab = "Miles per Gallon",
     breaks = 10)

## Box plot

boxplot(mtcars$mpg,
        main = "Boxplot of MPG",
        ylab = "Miles per Gallon")

Skewness and Kurtosis using formula

What did you observe about the shape of distribution of mileage of cars? Let us confirm it using statistics.

x <- mtcars$mpg
n <- length(x)
mu3 <- sum((x - mean(x))^3)/n
mu4 <- sum((x - mean(x))^4)/n
# Sample skewness
skewness <- mu3/sd(x)^3
cat("Skewness of x =", skewness, "\n") # Don't care about making it pretty? just type 'skewness' instead
## Skewness of x = 0.610655
# Sample excess kurtosis
kurtosis <- mu4/sd(x)^4
cat("Kurtosis of x =", kurtosis, "\n")
## Kurtosis of x = 2.627234

Scatterplot

Scatter plot is a very easy way to visualize associations between two variables

plot(mtcars$wt, mtcars$mpg,
     main = "MPG vs Weight",
     xlab = "Weight (1000 lbs)",
     ylab = "Miles per Gallon")

Correlation

In the scatter plot we can observe a negative association between weight and mileage of the cars we can confirm this by calculating the correlation between them.

cor(mtcars$wt, mtcars$mpg)
## [1] -0.8676594
